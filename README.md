# 🎓 EduConnect - Community Learning Platform

A fully responsive multi-page web application built for the Web Application Development Project Assignment.

## 🌐 Live Site
[View Live Site](https://txny537.github.io/EDUcommunityplatform)

## 📄 Pages
| Page | Description |
|------|-------------|
| Home | Landing page with hero, courses and testimonials |
| About | Mission, team profiles and core values |
| Courses | Full course catalogue with live search and filter |
| Blog | Articles and community stories |
| Gallery | Photo gallery with category filter |
| FAQ | Accordion FAQ with live search |
| Contact | Contact form with JavaScript validation |

## 🛠️ Technologies Used
- HTML5
- CSS3
- JavaScript ES6+
- Bootstrap 5.3
- Git & GitHub

## ✨ Interactive Features
1. Live search and category filter on Courses page
2. Dark/Light mode toggle saved to localStorage
3. Gallery category filter
4. FAQ live search
5. Contact form with JavaScript validation
6. Back to top button
7. Auto active navbar link highlighter
8. Navbar scroll shadow effect
