🎓 EduConnect

Web Application Development Project — Anton 2026

A fully responsive, multi-page community educational platform built with HTML, CSS, JavaScript and Bootstrap.

📌 What is this project?

This project is a fake but realistic website for an educational platform called EduConnect. Think of it like a mini version of Udemy or Coursera, but built for a Kenyan community.

The goal was to build a fully responsive multi-page web application with realistic content using core front-end technologies and professional version control.

📄 The 7 Pages

🏠 index.html

Homepage — Hero section, featured courses, stats bar, testimonials and footer.

👥 about.html

About Us — Mission statement, team profiles, stats counter and core values.

📚 courses.html

Courses — Full catalogue with live search, category filter and progress bars.

📝 blog.html

Blog — Featured article, article cards, newsletter signup section.

🖼️ gallery.html

Gallery — 9 real photos with category filter for Events, Workshops, Graduation.

❓ faq.html

FAQ — Bootstrap accordion with 8 questions and live search filter.

📞 contact.html

Contact — Form with JavaScript validation, contact info cards and Google Maps embed.

🛠️ Technologies Used

📄 HTML5

Structure of every page. Semantic tags like nav, section, footer, article used throughout.

🎨 CSS3

Custom styling in style.css. Includes animations, dark mode, hover effects and responsive tweaks.

⚡ JavaScript ES6+

All interactivity in main.js. Uses modern JS features like arrow functions, template literals and localStorage.

🅱️ Bootstrap 5

Loaded via CDN. Used for navbar, grid, cards, badges, buttons, accordion, progress bars and forms.

📦 Git

Used for version control. Every feature and fix was committed with a meaningful message.

🐙 GitHub

Remote repository at github.com/txny537/EDUcommunityplatform. Shows full commit history and code.

✨ Interactive Features

1. 🔍 Live Course Search

Filters course cards in real time as user types. Uses getElementById and addEventListener input.

2. 🏷️ Category Filter

Filter buttons show/hide course cards by category using data-category attributes.

3. 🌙 Dark Mode Toggle

Switches between light and dark mode. Saves preference to localStorage so it persists after refresh.

4. 🖼️ Gallery Filter

Filter buttons on gallery page show/hide cards by Events, Workshops or Graduation category.

5. ❓ FAQ Search

Searches through both questions and answers in real time. Shows no results message if nothing found.

6. 📋 Form Validation

Validates name, email, subject and message before submit. Shows green/red borders and success/error messages.

7. ⬆️ Back to Top Button

Fixed button appears after scrolling 300px. Smoothly scrolls back to top when clicked.

8. 🔗 Auto Active Navbar

Automatically highlights the correct navbar link based on which page the user is on.

9. 🔢 Counter Animation

Numbers on About page count up from 0 to their target value when the page loads.

10. 🌑 Navbar Scroll Shadow

Adds a shadow to the navbar when user scrolls down. Removed when back at the top.
